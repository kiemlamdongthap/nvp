Du an Frontend:

frontend-root/
├── public/
├── src/
│   ├── assets/             # Hình ảnh, icons, fonts...
│   ├── components/         # Các thành phần tái sử dụng
│   │   ├── AddProductModal.js         (Thêm gỗ vào CS đã có)
│   │   └── AddBreedingProductModal.js (Thêm ĐV vào CS đã có)
│   ├── contexts/           # Quản lý State (AuthContext, ProjectContext...)
│   ├── data/               # Dữ liệu tĩnh hoặc cấu hình
│   ├── layouts/            # Layout chung (Header, Sidebar, Footer)
│   ├── pages/              # Các trang chính trong ứng dụng
│   │   ├── KhaiBaoCoSoPage.js         (/khai-bao)
│   │   ├── KhaiBaoCoSoDongvat.js      (/khai-bao-dongvat)
│   │   ├── RegisterManageSub1Page.js  (/wood-farms)
│   │   ├── WoodDetail.js              (/woods)
│   │   ├── WoodEditPage.js            (/edit-wood)
│   │   ├── RegisterManageSub2Page.js  (/breeding-farms)
│   │   ├── FarmDetail.js              (/farm-details)
│   │   ├── FarmEditPage.js            (/edit-farm)
│   │   ├── MasterProductListPage.js   (/google-maps - Tổng hợp gỗ)
│   │   ├── MasterAnimalListPage.js    (/breeding-farms - Tổng hợp ĐV)
│   │   ├── GoogleMapsPage.js          (Bản đồ CS gỗ + ĐV)
│   │   └── userListpage.js            (Danh sách người dùng)
│   ├── utils/              # Các hàm bổ trợ (format date, validate...)
│   ├── api.js              # Cấu hình Axios/Fetch gọi Backend
│   ├── App.js              # File điều hướng chính (Routes)
│   ├── Dashboard.css       # Style riêng cho Dashboard
│   ├── index.css           # Global style
│   └── index.js            # Entry point của ứng dụng
├── .env                    # Biến môi trường (API_URL...)
├── .gitignore              # Bỏ qua khi push lên Git
└── package.json            # Quản lý dependencies
